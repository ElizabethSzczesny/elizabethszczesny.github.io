Project Overview

This project was creating a delivery tracking flow for a fictional logistics company. When tracking a package, especially internationally, consumers prefer as much information, regarding the whereabouts and timeline of delivery, as possible. The aim of this application is to efficiently provide these details in a reader-friendly format.
